# 部分功能截图展示，需要自己部署本地模型，可以选用1到2B的国产模型
## ✨ 功能特性
- 支持一键导入配置，自动运行。
- 毫秒级识别
- 用鼠标选取单词自动识别
- 支持中英文互译

## 🚀 快速开始


### 1. 环境要求
请确保你的电脑上已经安装了 Python 3.8 或以上版本。

### 2. 安装步骤
` ` `克隆本项目到本地，并安装依赖包：
bash

cd 你的仓库名
